## 🗺️ MAP OBJECTIVE COLOR KEY

| Color | Type | Priority |
|---|---|---|
| 🟣 **MAGENTA/PURPLE** | Primary objective | **WIN/LOSE — Always prioritize** |
| ⚫ **BLACK** | Secondary objective | Important for supply/radio chain |
| 🔵 **BLUE** (your color) | Friendly-held | Defend and supply |
| 🔴 **RED** (enemy color) | Enemy-held | Attack and capture |

> **Red = Russia. Blue = NATO. Always.**

## 🩸 MEDEVAC SEQUENCE

`DRAG → TOURNIQUET → BANDAGE → SALINE → EPI → MORPHINE → Hospital`

**Never treat in the open. Drag to hard cover first.**

## 📻 BREVITY CODES

| Say | Means |
|---|---|
| `Contact [Dir + Distance]` | Enemy spotted and engaging |
| `TIC` | Troops in Contact — pinned down |
| `Winchester` | Out of ammo |
| `RTB` | Returning to Base |
| `Danger Close` | Friendlies within 100m of fire mission |
| `Frag Out` | Say BEFORE throwing a grenade |
| `Litter` | Casualty needs medical attention |
| `Check Fire` | STOP FIRING — friendly fire risk |

## ⚓ LOGISTICS LOOP

`MOB → Load at Supply Depot → Drive to FOB → Unload → Repeat`

**Build a relay chain: MOB > mid-FOB > frontline FOB**

## 💥 CALL FOR FIRE FORMAT

```
1. "Command, Recon-1, request Fire Mission, over."
2. "Grid 084 125." (or landmark)
3. "One truck and a squad digging a FOB."
4. "HE, three rounds, fire for effect."
5. Adjust: "Add 50, Left 20."
6. "Target destroyed. End of mission."
```

## 🪖 ROLE AT A GLANCE

| Role | Primary Job | Key Item |
|---|---|---|
| **Assault** | Push objectives, clear buildings | C4 + AT weapon |
| **MG / AG** | Base of fire from elevation | AG carries all MG ammo |
| **Anti-Armor** | Kill vehicles + aircraft | Work in pairs — ammo bearer required |
| **Medic** | Keep squad combat-effective | Never lead the stack |
| **Recon** | Feed intel + call for fire | Light kit, long-range radio |
| **Armor** | Combined arms, never solo | Repair Tool mandatory |
| **Logistics** | Supply chain, FOB building | Biggest truck available |

## 🚫 INSTANT BAN LIST

- Wearing **enemy faction gear** (helmets, vests, uniforms, boots)
- **Dumping supplies** on the ground to drain depots
- Racist, hateful, or offensive language
- Firing into enemy **Main Base**
- Helicopter ramming
- Using or sharing **exploits**

> WCS bans are **network-wide** — banned on one server = banned on all 16+
