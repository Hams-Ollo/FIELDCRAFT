## 📚 ABOUT THIS LIBRARY

This is a living field manual for Arma Reforger — built by veterans and experienced players, written for everyone from first-session recruits to seasoned squad leaders.

The documentation is grounded in real US Army doctrine — publicly available field manuals that have been battle-tested over decades. We translate the **principles** directly into WCS game mechanics. The **why** is real. The **how** is adapted to the game. The result is a library that makes you a genuinely better player while teaching you the same fundamentals that real military units use.

This is not a tryhard document. It is designed to be approachable, consumable, and fun to read — while being tactically credible. If you served, you will recognize the doctrine. If you did not, you will understand why it works.

## �📖 DOCTRINE FOUNDATION

This library uses two primary US Army field manuals as its doctrinal backbone. Both are publicly available:

**ATP 3-21.8 — Infantry Platoon and Squad**
The foundational manual for how small infantry units fight. Covers movement formations, fire and movement principles, react to contact drills, ambush procedures, and defensive operations. Every tactical SOP in this library traces back to these principles.

**FM 6-30 / ATP 3-09.30 — Observed Fire / Tactics for Fire Support**
The observer's manual for calling and adjusting indirect fires. The six-element call for fire format, adjustment procedures, target reference point methodology, and observer post selection all derive from this manual.

The adaptation principle throughout this library: **the doctrine tells you why. The game mechanic tells you how.** We never drop a doctrine concept just because it is complex — we translate it until it is usable.

See `DOCTRINE_SOURCES.md` for full references and links.
