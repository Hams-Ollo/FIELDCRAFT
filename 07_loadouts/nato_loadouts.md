# NATO LOADOUTS
## WCS Milsim — Official Gear Specifications

> **Source:** WCS MILSIM Loadout Handbook (Official Publication)
> **Faction:** NATO (MultiCam, Tan, or Coyote color scheme throughout)
> **Revision:** See ROADMAP.md for version history

---

## OVERVIEW

This document lists the authoritative in-game gear specifications for all NATO roles. Adhere to these exactly — do not substitute items not listed. Target weight limits are hard constraints.

**Medical baseline for all standard roles:** 5x Bandages, 3x Tourniquets, 3x Ammonium Carbonate

**Universal equipment (all roles carry unless specified otherwise):**
- Map, Compass, Garmin Tactix Bravo Watch
- AN/PRC-152 Radio (short-range)
- AN/PSN-13 DAGR (GPS)
- Vector 21 Binoculars
- M1967 Entrenching Tool
- Wirecutters
- Any NVGs
