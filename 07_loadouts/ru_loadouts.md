# RU LOADOUTS
## WCS Milsim — Official Gear Specifications (Russian Faction)

> **Source:** WCS MILSIM Loadout Handbook (Official Publication)
> **Faction:** Russian (EMR Digital Flora or Olive Drab color scheme throughout)
> **Revision:** See ROADMAP.md for version history

---

## OVERVIEW

This document lists the authoritative in-game gear specifications for all Russian faction roles. Adhere to these exactly — do not substitute items not listed.

**Medical baseline for all standard roles:** 5x Bandages, 3x Tourniquets, 3x Ammonium Carbonate

**Universal equipment (all roles carry unless specified otherwise):**
- Map, Compass, Garmin Tactix Bravo Watch
- R-187-P1 "Azart" Radio
- 15Ts8009 "Orion" (GPS)
- PDU-4 Binoculars
- MPL-50 Entrenching Tool
- Wirecutters
- Any NVGs

> **Key difference from NATO:** Russian forces use R-187-P1 Azart radios (instead of AN/PRC-152) and Orion GPS (instead of DAGR). Same procedures apply. Russian roles are a subset of NATO — no JTAC, SNOT, Mortarman (Ammo Bearer), Aviation, or Platoon-level leadership roles in RU loadouts.

## FACTION COMPARISON QUICK REFERENCE

| Element | NATO | Russia |
|---|---|---|
| **Base rifle caliber** | 5.56x45mm | 5.45x39mm |

| **AT system** | MK153 SMAW / M72A3 LAW | RPG-7V1 |
| **Radio** | AN/PRC-152 | R-187-P1 Azart |
| **GPS** | AN/PSN-13 DAGR | 15Ts8009 Orion |
| **Optics** | Vector 21 | PDU-4 |
| **Uniform** | MultiCam / Tan / Coyote | EMR Digital Flora / Olive Drab |
| **Medical tiers** | CLS → SM → CM → PM | CLS → SM only |
